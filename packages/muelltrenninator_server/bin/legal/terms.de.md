## 1. Geltungsbereich

## 2. Dienstbeschreibung

Mülltrenninator ist ein kostenloser Webdienst, der Abfallbilder mithilfe künstlicher Intelligenz klassifiziert. Sie laden ein Foto hoch und erhalten eine KI-gestützte Vorhersage, die angibt, um welche Art von Abfall es sich handelt und ob es sich um Restmüll handelt. Der Dienst erfordert keine Registrierung und kein Benutzerkonto. Es werden keine Bilder vom Server gespeichert.

## 3. Keine Registrierung erforderlich

Mülltrenninator bietet keine Benutzerkonten an und erfordert keine solchen. Es gibt keine Registrierung, keine Anmeldung und keine Authentifizierung. Der Dienst steht zur anonymen Nutzung offen.

## 5. Nutzerinhalte

- Sie behalten das Eigentum an von Ihnen hochgeladenen Bildern.
- Hochgeladene Bilder werden an ein KI-Modell-Backend zur Klassifizierung weitergeleitet und vom Mülltrenninator-Server **nicht gespeichert**. Der Server fungiert als Proxy, der das Bild zur Vorhersage weiterleitet und das Ergebnis zurückgibt.
- Wir beanspruchen keine Lizenz oder Rechte an Ihren hochgeladenen Bildern über die vorübergehende Verarbeitung hinaus, die zur Bereitstellung des Klassifizierungsergebnisses erforderlich ist.

## 6. Bildverarbeitung

Wenn Sie ein Bild zur Klassifizierung hochladen:

1. Das Bild wird an ein KI-Modell-Backend (Gradio-basiert) zur Vorhersage weitergeleitet.
2. Das Modell analysiert das Bild und gibt ein Klassifizierungsergebnis zurück.
3. Das Bild wird vom Mülltrenninator-Server **nicht gespeichert, aufbewahrt oder für Training verwendet**.
4. Das Modell-Backend kann das Bild vorübergehend im Speicher oder auf der Festplatte halten; es wird nach Abschluss der Vorhersage nicht aufbewahrt.
